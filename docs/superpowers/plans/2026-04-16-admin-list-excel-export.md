# Admin List Excel Export Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `.xlsx` export support for `使用日志`、`额度流水`、`审计日志`, using the current committed query, honoring current visible columns for usage logs, and enforcing the approved `2000`-row cap.

**Architecture:** Keep the export pipeline backend-generated. The frontend only sends the current committed filters plus visible usage-log column keys, while the backend reuses existing list-query semantics, applies the hard cap, maps records into export rows, and streams a native `.xlsx` file. Shared responsibilities are split into a small Excel writer service, capped export query helpers, and page-specific frontend download wiring.

**Tech Stack:** Go 1.22, Gin, GORM, `github.com/xuri/excelize/v2`, React 18, Semi UI, Bun, `node:test`, `go test`

```go
type ExcelFileSpec struct {
	FileNamePrefix string
	SheetName      string
	Headers        []string
	Rows           [][]string
}

func BuildExcelFile(spec ExcelFileSpec) (string, []byte, error) {
	file := excelize.NewFile()
	sheet := spec.SheetName
	defaultSheet := file.GetSheetName(0)
	file.SetSheetName(defaultSheet, sheet)

	for idx, header := range spec.Headers {
		cell, _ := excelize.CoordinatesToCellName(idx+1, 1)
		file.SetCellValue(sheet, cell, header)
	}
	for rowIndex, row := range spec.Rows {
		for colIndex, value := range row {
			cell, _ := excelize.CoordinatesToCellName(colIndex+1, rowIndex+2)
			file.SetCellValue(sheet, cell, value)
		}
	}

	buffer, err := file.WriteToBuffer()
	if err != nil {
		return "", nil, err
	}
	return fmt.Sprintf("%s_%s.xlsx", spec.FileNamePrefix, time.Now().Format("2006-01-02_15-04-05")), buffer.Bytes(), nil
}
```

export const extractDownloadFilename = (headers, fallback = 'export.xlsx') => {
  const disposition = headers.get('content-disposition') || '';
  const match = disposition.match(/filename=\"?([^\";]+)\"?/);
  return match?.[1] || fallback;
};

export const downloadExcelBlob = async ({ url, payload, fallbackFileName }) => {
  const response = await API.post(url, payload, { responseType: 'blob' });
  const fileName = extractDownloadFilename(response.headers, fallbackFileName);
  const blob = new Blob([response.data], {
    type: 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',

};
